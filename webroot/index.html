Web vulnerability examples These pages contain demos of the web vulnerabilities shown in lecture. Demos are grouped roughly by the type of vulnerability and how they were shown in lecture. See the source files in the webroot directory to see how the pages work and for examples of exploit code.

Demo pages:
  • 01-client-csrf: Client-side controls, CSRF
  • 02-sqli: SQL injection
  • 03-xss: Cross Site Scripting (XSS)
  • 04-xss-02: XSS + SQLI Sanitization examples
  • 05-eval: Server-side remote code execution
